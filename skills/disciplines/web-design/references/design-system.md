# Design-system reference

Record:

- visual register;
- hierarchy principles;
- typography roles;
- color roles and contrast obligations;
- spacing character;
- shape and border language;
- image and icon direction;
- motion purpose and limits;
- responsive content priorities;
- repeated composition patterns;
- deliberate exceptions.

Prefer positive rules that describe the desired result. Use prohibitions only for hard constraints and pair each with the intended alternative.
